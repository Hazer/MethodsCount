Android Methods Count

A simple Studio/IntelliJ plugin that parses your Android library dependencies and shows the methods count as an handy hint.

For more information, you can simply tap on the blue circle to the left and you will land on the appropriate library page of this website.


Installation
  1. Open Android Studio / IntelliJ IDEA preferences
  2. Select 'Plugins'
  3. Select 'Browse repositories...'
  4. Select 'Android Methods Count' (or type some keywords in the search box)
  5. Press install and restart the IDE once the installation completes

Alternatively, you can download the plugin archive from here.
